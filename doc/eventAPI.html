    chart api

    获取图表坐标点数组接口

    /event/chart get

    get

    得到图表上的坐标点对象数组

    get /event/chart

    得到图表上的坐标点对象数组

    • Request
    • Response

    Query Parameters

    • source: required (string)

      来源：百度贴吧

      Example:

      百度贴吧
    • data: required (string)

      数据类型：跟帖量 或 发帖量

      Examples:

      example:

      跟帖量

      example1:

      发帖量
    • beginTime: (string)

      开始日期：格式 日/月/年

      Example:

      05/01/2017
    • endTime: (string)

      结束日期：格式为 日/月/年

      Example:

      05/01/2017

    HTTP status code 200

    获取图表成功

    Body

    Media type: application/json

    Type: array of items

    Items: items

    • x: required (string)

      横坐标

    • y: required (integer)

      纵坐标

    Example:

    [{"x":"2017-05-01","y":6},{"x":"2017-05-02","y":15},{"x":"2017-05-03","y":0}]

    HTTP status code 400

    请求失败：参数不合法

    Body

    Media type: application/json

    Type: string

    Example:

    参数不合法

    HTTP status code 500

    请求失败：服务器内部错误

    Body

    Media type: application/json

    Type: string

    Example:

    服务器内部错误

      传入指定的记录人与事件编号，进行归集操作。

      post

      post /event/dailyEvent/{id}/{recorder}/collect

      • Request
      • Response

      URI Parameters

      • id: required (integer)

        日常检测事件的id

        Examples:

        example:

        1

        example1:

        2
      • recorder: required (string)

        记录人名称（名字长度不超过10）

        Example:

        威尔史密斯
